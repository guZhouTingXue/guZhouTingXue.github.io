# 闭店
预打烊：闭店前3小时，可以依据实际运营状况，预先完成部分打烊工作（清理器具）

整体流程：
- 器具清洁
- 设施设备清洁
- 闭店工作
## 1. 器具清洁
- 去除残渣
- 配置清洁液
- 在清洁液中清洗
- 温水冲洗
- 消毒：浸泡在消毒水中 5min，巧克力桶浸泡（10min？）
- 再次温水冲洗
- 擦拭

### 酱料桶（预调液桶）
组成部分：
- 桶身
- 桶盖
- 外部泵头
- 底部软管（软）
- 内部吸管（硬）：上粗下窄
- 弹簧
- 垫片
- 三棱坐（不知道叫啥）

组装顺序：底部吸管连接内部吸管下方-》从硬吸管上方放入垫片-》三棱坐底部朝下放入硬吸管-》放入弹簧-》桶盖-》外部泵头向下按压并拧紧-》连接桶身

清洗时间：10min

组装时使用一次性手套

### 咖啡机
**日清**
咖啡豆仓区分：
- 浅烘：绿色，在最左侧
- 耶加雪菲：蓝色，中间
- 普通咖啡豆：左侧
- 深烘：红色，中间

粘贴效期时，普通咖啡豆不用补打效期，其余类型咖啡豆补打效期。

照片：
- 蒸奶棒浸泡：30ml 牛奶系统清洁液，500ml热水，浸泡10分钟
- 擦拭豆仓：佩戴一次性手套
- <font color="#ff0000">红色</font>长柄刷清洁冲泡器周边
- 蒸奶棒擦拭：红色毛巾、一次性手套

蒸奶棒清洗完成安装回咖啡机后需要高温吹气将棒内残留的清水吹走（吹干），重复3次高温蒸汽动作，每次动作时间不少于3s。
