# 原料预置
## 茶汤
准备：
干净的冷水壶、镊子
1-2包使用冷水壶，3-5包使用萃茶机

| 步骤  | 内容      | 计量                                           | 说明                                                                 |
| --- | ------- | -------------------------------------------- | ------------------------------------------------------------------ |
| 1   | 打印并粘贴效期 |                                              | 预置物料-》对应物料名称-》数量-》确定。多打印1个标签用于填写过期时间，使用红色水笔填写过期时间-实际过期时间前15分钟（大致）。 |
| 2   | 添加茶包    |                                              | 如需开封新的茶包包装，在原包装开封界面打印茶包包装开封效期                                      |
| 3   | 接取热水    | 阿萨姆红茶：1包/600ml；2包/1400ml<br>茉莉：1包/；2包/1400ml |                                                                    |
| 4   | 搅拌后开始计时 | 阿萨姆：<br>茉莉：                                  |                                                                    |
| 5   | 取出茶包    |                                              | 标准使用吧勺取处茶包，实际用镊子夹取<br>标准考虑镊子可能戳破茶包，导致茶叶混入饮品被当作异物                   |
| 6   | 取冰      |                                              | 泡制2包添加的冰块就大概3满16oz铲冰                                               |
| 7   | 搅拌至冰块融化 |                                              |                                                                    |
