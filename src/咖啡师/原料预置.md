# 原料预置

## 效期打印
效期有两种：
1. 正在使用的材料的效期：材料在容器内，已经处理好了，在出餐时需要使用。效期粘贴在容器上（冷水壶、份数盒。。。）
2. 未使用的材料的效期：材料在包装内，未被处理。效期粘贴在包装袋上

- 茶汤：预置
- 奶盖、抹茶：预置，按实际预置的量打印效期。如抹茶最大预置量为200，然后实际预置了400的抹茶，就需要打印两次200的效期
- 罐头：容器内

1. 茶包开封需要打印原包装开封效期，然后粘贴到茶包包装上
2. 罐头如需补充无需额外打印容器内效期，直接开封添加。如当前添加的材料和原容器内材料的效期不一致，需要重新打印容器内效期，并更换/清洗容器，替换效期。比如：有效期是2天，当天开封使用完，再次添加时直接添加。 但是如果当天没用完，第二天需要添加，那么需要重新打印容器内效期。 

确保效期和当前使用的材料的效期一致
## 茶汤
准备：
干净的冷水壶、镊子
1-2包使用冷水壶，3-5包使用萃茶机

| 步骤  | 内容      | 计量                                              | 说明                                                                 |
| --- | ------- | ----------------------------------------------- | ------------------------------------------------------------------ |
| 1   | 打印并粘贴效期 |                                                 | 预置物料-》对应物料名称-》数量-》确定。多打印1个标签用于填写过期时间，使用红色水笔填写过期时间-实际过期时间前15分钟（大致）。 |
| 2   | 添加茶包    |                                                 | 如需开封新的茶包包装，在原包装开封界面打印茶包包装开封效期                                      |
| 3   | 接取热水    | 阿萨姆红茶：1包/600ml；2包/1400ml<br>茉莉：1包/600；2包/1400ml |                                                                    |
| 4   | 搅拌后开始计时 | 阿萨姆：10min<br>茉莉：8min<br>龙井：6min                 |                                                                    |
| 5   | 取出茶包    |                                                 | 标准使用吧勺取处茶包，实际用镊子夹取<br>标准考虑镊子可能戳破茶包，导致茶叶混入饮品被当作异物                   |
| 6   | 取冰      | 1包/900ml；2包/1800ml                              | 泡制2包添加的冰块就大概3满16oz铲冰                                               |
| 7   | 搅拌至冰块融化 |                                                 |                                                                    |

热水-》茶-》冰-》搅拌至冰块溶解
加盖常温保存，当日闭店废弃

## 预置物料报损
红茶、茉莉
路径：单屏机-》库存管理-》-》预置物料-》报损-》调整库存，输入剩余茶汤容量。单个茶包最大预置900ml，剩余含量不能大于最大预置量。-》调整原因：过期销毁-》确认

1. 开早时会弹窗提示预置物料：点击制作
2. 营业期间手动预置（系统不会弹窗提醒）
3. 按小于等于系统推荐的数量进行预置
