# 开早
>[!info]
>开早是指在营业前30分钟之内，完成开店工作

整体流程：
1. 到店准备
2. 物料准备
3. 清洁准备
## 1. 到店准备
**设备开启**
运营设备
- 单屏机：
- 小票机
- 标签机
- 扫码器

生产设备
- 咖啡机
- 开水机
- 料理机

**消毒用品准备**
1. 配置消毒水
原料：容杯、消毒粉、温水、吸管、分配器
2. 抹布消毒：完全浸泡

**器具摆放**

4. 运冰
