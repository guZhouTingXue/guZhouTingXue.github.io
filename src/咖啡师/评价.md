# 评价
**缺点**

| 序号  | 内容       | 说明                                                                                                                                               |
| --- | -------- | ------------------------------------------------------------------------------------------------------------------------------------------------ |
| 1   | 克扣工时     | 比如排班：13：30 ~ 22：30， 中间有两次休息共1个半小时，但实际只休息不到半小时。多的时间都是打白工。                                                                                         |
| 2.  | 学习成本     | 每周都会上新品然后经常性的运营事物调整（制作标准、清洁标准。。。）这些东西都要记住，但是花的都是自己的时间，上班时就只有干活。尤其是前期：入职的准备、培训、配方记忆、门店熟悉。。。                                                       |
| 3   | 标准严格     | 每半小时换一次红色抹布；双数整点洗一次手；每隔3小时换一次消毒水。。。基本上没有多余休息时间，相比工厂流水线，可以看作是“活动的流水线“                                                                             |
| 4   | 个人发展空间有限 | 两种晋升路线，大部分咖啡师-》值班-》副店长（全职）-》店长，这就到头了                                                                                                             |
| 5   | 排班       | 一个店只有店长、副店长是全职，他们随时准备上班，其他人都是兼职。如果控制工时，那么排班可能比较混乱，会根据当前实时单量让你提前上班或推迟下班。原本是下午3点上班，正常两点多准备上班。因为可能出现提前的情况，要在1点多就准备好。这样实际是侵占了个人休息时间，人虽然没到店里，但是心已经到了。 |
| 6   | 规则制定虎头蛇尾 | 1. 因为发现有员工手有伤口未处理、未佩戴手套，要求必须佩戴胶皮手套。现在不是必须<br>2. 预置物料报损。首先这个功能感觉就不完善，现在茶汤到期直接废弃，奶盖闭店时也是直接倒掉，没有报损过。                                                |
| 7   | 入职流程繁琐   | 需要办理健康证，以及完成入职前的线上培训，个人购买工服                                                                                                                      |

瑞幸目前有3w+门店，在职的员工大概有7.4w（来源网络），包含了全职以及兼职。
干了瑞幸之后，不可能再让我花钱买瑞幸了。虽然之前基本上也没喝过瑞幸（大概2、3次）。第一次喝瑞幸应该还是在长沙的时候，记忆中当时还是20左右一杯美式。对于我来说还是太高端了。喝的比较多的奶茶就是茶颜、蜜雪还有古茗。现在奶茶的价格被卷下来，普遍最低9.9一杯，但相比于瓶装饮料还是贵。营养方面，奶茶包含的原料和瓶装饮料基本无异。食品卫生方面，奶茶不可能比真正的流水线产品更卫生。最重要的是瑞幸的工作让我对奶茶店-这个环境过敏

**其他**
- 只能根据完整的手机号码搜索订单（不能查询尾号且没有任何提示）
- 点击取餐，不能选择区间，只能一个一个点
- 咖啡机的豆仓
	- 装卸费力：底部的封口片安装是有正反的。应该安装错误，然后强行固定，导致锁止拉杆不能很好地移动，解除锁定的时候封口片卡住，出不来。每次取豆仓的时候都十分费力。
	- 豆仓能不能设计为倾斜的？进豆口在前面，用着用着口子附近就空了，但是后面的豆子不会自己往前面落，要手动把后面的豆子扒到前面。
