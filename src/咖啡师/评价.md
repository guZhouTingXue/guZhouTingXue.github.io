# 评价
**缺点**

| 序号  | 内容       | 说明                                                                                         |
| --- | -------- | ------------------------------------------------------------------------------------------ |
| 1   | 克扣工时     | 比如排班：13：30 ~ 22：30， 中间有两次休息共1个半小时，但实际只休息不到半小时。多的时间都是打白工。                                   |
| 2.  | 学习成本     | 每周都会上新品然后经常性的运营事物调整（制作标准、清洁标准。。。）这些东西都要记住，但是花的都是自己的时间，上班时就只有干活。尤其是前期：入职的准备、培训、配方记忆、门店熟悉。。。 |
| 3   | 标准严格     | 每半小时换一次红色抹布；双数整点洗一次手；每隔3小时换一次消毒水。。。基本上没有多余休息时间，相比工厂流水线，可以看作是“活动的流水线“                       |
| 4   | 个人发展空间有限 | 两种晋升路线，大部分咖啡师-》值班-》副店长（全职）-》店长，这就到头了                                                       |
