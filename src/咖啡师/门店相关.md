# 门店相关

1. 物品摆放
1.1 开封物料冰箱
根据物料使用频率以及搭配放置物料
左上固定放置：椰浆、牛乳、牛奶及黄油
右下角：芒果、椰奶冻
右上：马斯卡彭、太妃、燕麦奶、红豆
1.2 后吧台（果茶制作）
- 一桶直饮水
- 一桶茉莉（从萃茶机萃取的茶汤中分配，注意效期）
- 一个冰摇壶
- 一个量杯
1.3 冲洗池
放置一个吧勺
1.4 开水机
开水机下放置一个奶缸以及一个吧勺，吧勺搅拌后使用奶缸内剩余开水冲洗，倒入咖啡机接水槽
1.5 剪刀
剪刀放置在茶桶架下，使用后用红色毛巾擦拭。剪下的垃圾放入围裙口袋内，不要丢弃在冲洗池（保持冲洗池干净）
1.6 储冰车
储冰车固定在冲洗池，避免污染消毒池

2. 高峰期注意事项
2.1 物料准备
在高峰期前准备齐全制作物料
准备时间：周一~周五10点前，周末：11点前
2.2 高峰期结束
高峰期结束后清理台面，收拾物料，保持台面卫生。

3. 接单岗职责
接单岗在贴杯后根据订单整理台面，将不需要的物料及时放入冰箱。

4. 扣分注意事项
4.1 营业中
- 盖完盖子后扫码，并将加盖后的饮品放置在出餐区域
- 吧勺冲洗后使用
4.2 打烊
- 咖啡机清洁冲泡器
- 消毒池内物品完全浸泡在消毒水中，在浸泡时间超过5分钟后进行清洗
- 单屏机关机
