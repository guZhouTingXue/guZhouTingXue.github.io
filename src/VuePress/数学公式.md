# 数学公式
1. 安装
[KaTeX – The fastest math typesetting library for the web](https://katex.org/)：在网页渲染使用LaTex语法编辑的公式 

``` bash
npm i -D katex
```

2. 启用
``` ts title="theme.ts"
export default hopeTheme({
  markdown: {
    math: {
      type: "katex", //[!code highlight]
    },
  },
});
```

3. 使用
:::preview
$$
\begin{vmatrix}a & b\\
c & d
\end{vmatrix}=ad-bc
$$

这是文本行内的公式$e^{2i\pi} = 1$
:::

>[!warning]
>inline 语法\$ 和公式符号之间不能有空格，公式内可以有空格

如：
:::preview
示例：$ y=3$ 、 $y=3 $ 、 $ y=3 $ 都不行
$y = 3$ 可以
:::
## 其他
1. inline 公式 可以使用\ 进行转义（即\ + $ 不再是LaTex 语法）
:::preview
对$ 进行转义，这是 行内语法：$y = x+1$ , 转义：\$y=x+1$
:::
