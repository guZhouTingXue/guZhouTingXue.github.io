# 更新

## 查看当前版本
``` bash
npm list vuepress-theme-hope
npm list @vuepress/bundler-vite
npm list vuepress
```
包的名称可以在`package.json`中查看

## 查看历史版本
命令：
``` bash
npm view vuepress-theme-hope versions --json
```
结果：
``` 
[
  "0.0.1-alpha.0",
  "0.0.1-alpha.1",
...
  "2.0.0-rc.97",
  "2.0.0-rc.98"
]
```

## 安装（更新）
``` bash
npm i @vuepress/bundler-vite@2.0.0-rc.26
npm i vuepress@2.0.0-rc.26
```
在包名后添加`@版本名称`指定更新（安装的版本）
