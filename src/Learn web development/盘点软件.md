# 盘点软件
## 软件需求
-  能够按照：物料类型，如奶基、果汁、轻食、咖啡；存放位置，如轻食冰箱、果汁冰箱、预置物料冰箱等进行分类显示
	-  能够编辑分类
- 能够按照指定过滤条件进行过滤：
过滤的条件可以有：数量、存放位置、种类。。。，条件可以进行&、|| 组合
如放置在仓库的咖啡，这里有两种分类类型，位置-仓库。物料类型-咖啡。

- 在确定了分类之后可以编辑物料类型
- 可以查看历史盘点数据
	- 可以选择历史数据进行比较，查看差异
	- 选择具体某一天的历史数据
	- 查看指定区间范围内的历史数据以及变化趋势
	- 当天盘点数据可显示昨日数据 并 能够直接采用
- 可以分步进行盘点：如闭店前1小时 耶加雪菲咖啡停售，此时可以盘点耶加雪菲咖啡豆及其豆仓所在咖啡机的意式豆。闭店时清洗咖啡机，盘点剩余的意式。
对于意式可以划分为两步进行盘点：
1. 时间闭店前1小时
2. 闭店

- 盘点时输入带有包装、容器的整体重量：自动计算去壳后实际物料剩余含量
- 能够转换为实际盘点工单表格
- 实际库存 和 理论库存（系统计算的）存在差异，能够记录最终录入系统的库存数据
- 同一种物料可能有不同厂商、型号，能够提示需要进行区分，并显示产品的差异点 或 主要标志
