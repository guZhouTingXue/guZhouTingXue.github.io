# 命名

| 简称  | 全称              | 含义                                 | 示例                                                             |
| --- | --------------- | ---------------------------------- | -------------------------------------------------------------- |
| lhs | left hand side  | 出现在左侧。通常是函数有两个同类型的输出参数时，第一个参数名为lhs | bool compareIsbn(const Sales_item &lhs, const Sales_item &rhs) |
| rhs | right hand side | 第二个输出参数名为rhs                       |                                                                |
