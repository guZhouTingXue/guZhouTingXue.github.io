# Visual Studio Code

## 通用插件

| 名称                                            | 功能                                                                                                                         | 说明                                                     |
| --------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------ |
| [GitHub Repositories](#GitHub%20Repositories) | lets you quickly browse, search, edit, and commit to any remote GitHub repository directly from within Visual Studio Code. | 在vs code 上快速浏览github开源项目代码，文件临时缓冲。如果需要深入了解，需要将项目下载到本地。 |


### GitHub Repositories
**Getting started**
1. 输入命令：_Remote Repositories: Open Remote Repository..._
2. 输入github 开源项目网址
![|399x56](./attachments/Visual%20Studio%20Code.webp)

第一次执行需要登录github
3. 效果
![|298x202](./attachments/Visual%20Studio%20Code-1.webp)

4. 通过插件新增的 Remote Explorer 按钮查看浏览过的Remote Repositories 列表
![|398x275](./attachments/Visual%20Studio%20Code-2.webp)
