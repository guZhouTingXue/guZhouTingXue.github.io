# 输入法

## windows 进行手写输入
1. 鼠标右键点击右下角工具按钮区域，弹出菜单
工具按钮：
![](./attachments/输入法.webp)
弹出的菜单：
![](./attachments/输入法-1.webp)

2. 勾选弹出菜单中的 “显示触摸键盘按钮”，工具按钮区域会新增一个键盘工具按钮：
![](./attachments/输入法-2.webp)
3. 点击键盘按钮，弹出虚拟键盘
![|398x142](./attachments/输入法-3.webp)
4. 点击虚拟键盘中左上角的键盘按钮，设置为手写输入
![|399x137](./attachments/输入法-4.webp)
5. 通过鼠标输入
![|401x143](./attachments/输入法-5.webp)
