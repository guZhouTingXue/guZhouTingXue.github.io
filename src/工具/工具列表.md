## keyman
在主界面选择 IPA 语言，进入对应的下载页面，下载并安装程序
![](./attachments/工具列表-2.webp)

启动运行后， Input methods 中新增 IPA输入
![](./attachments/工具列表-3.webp)


**配置快捷键**
配置-》快捷键-》常规快捷键
选中 “打开语言切换器”，按住shift + space，进入快捷键配置界面
![](./attachments/工具列表-4.webp)

>[!note]
>这里配置的快捷键最好和系统设置中的 `Between Input Languages` 一致，否则可能会导致系统配置失效，无法切换输入法



**选项**
- 随windows 启动：需要运行keyman 程序，IPA输入法才有效果

**键盘布局**
通过字符序列：基本字符 + 修饰符（<、>、=） 组合成一个IPA基本字符
如输入 "i" + "=" 结果是 "ɪ"

## marktext
在复习文章 或 专注写作时，想要屏蔽干扰，或者说是减少不必要的信息输入，需要禅模式。ob 中虽然有 一个 Typewriter mode 插件，但是使用起来不尽人意。

:::tabs
@tab 普通模式
![|400x279](attachments/普通模式.gif)

@tab Typewriter mode
![|400x278](attachments/TypewriterMode.gif)
和普通模式的区别：
- 换行时界面会自动向上移动，使当前输入保持在相对中央的位置（即使界面能够显示全部内容）
- 切换/跳转 行时，界面也移动。
@tab focus mode
![|399x244](./attachments/工具列表-5.webp)
	突出当前编辑内容，淡化其他内容。
