# Microsoft Edge


## Extensions

| 名称                 | 功能                                                          |
| ------------------ | ----------------------------------------------------------- |
| Readline           | 阅读模式，相比于浏览器自带的阅读模式，Readline 可以设置是否显示图片                      |
| Tampermonkey       | 管理用户脚本（javascript 程序，用于修改网页，增强网页功能，如增加一个按钮）                 |
| Proxy SwitchyOmega | Manage and switch between multiple proxies quickly & easily |

### Readline
>[!note] 注意事项

  在该模式下保存为PDF时需要使用插件提供的工具栏中的 Print 功能，将 Destination 设置为Save as PDF。如果直接在页面中使用浏览器自带的Print（鼠标右键-》Print）只会保存当前显示的内容，无法将整个网页的内容保存为PDF
