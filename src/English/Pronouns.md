# 6. 代词 Pronouns
>[!info]
>代词是用来代替 ***名词*** 或 ***名词短语*** 的一类词
>**有些** 代词有性、数、格 和 人称的不同形式，所以作为名词的代用词，代词需同其先行词的性、数、人称一致，并须按其句中所起的 **语法作用**，用其相应 **格的形式**。
