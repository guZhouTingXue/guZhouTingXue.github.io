# 7.3 Voice
>[!info]
>语态是谓语动词的一种形式，用来表示句子的主语 和 谓语动词所表达的动作之间的关系。

| 类型                | 主语的作用        |
| ----------------- | ------------ |
| The Active Voice  | 动作的发出者 或 执行者 |
| The Passive Voice | 动作的承受者       |
