# 7. 动词 Verbs
>[!info]
>表示动作 或 状态。在句中 主要做谓语，有人称、数、时态、语态 和 语气的变化。不同时态 和 语态的动词组成的动词短语，还可做定语、状语、主语 和 宾语

## 分类
### 句法

| 序号  | 类型             | 示例  | 中文名称 |
| --- | -------------- | --- | ---- |
| 1   | Notional Verb  |     | 实义动词 |
| 2   | Link Verb      |     | 系动词  |
| 3   | Auxiliary Verb |     | 助动词  |
| 4   | Modal Verb     |     | 情态动词 |

### 语法结构
- 是否在句中作谓语-》受到主语的限定，有人称、数、时态、语态 和 语气的变化：限定动词 - Finite Verb
- 不能独立作谓语，必须与一定的助动词结合，一起构成复合谓语 或 充当其他句子成分-》不会受到主语（人称、数）的限制：非限定动词 - Non-finite Verb

#### Finite Verb
在表示语法概念中的限定时，finite的反义词是Non-finite。其他情况反义词是infinite

| example                                                 | predicate  | verb  |              |
| ------------------------------------------------------- | ---------- | ----- | ------------ |
| I am leaving for Beijing tommorrow.                     | am leaving | leave | 第一人称 + 现在进行时 |
| Character is the first and last word in success circle. | is         | is    | 第三人称 + 一般现在时 |
|                                                         |            |       |              |
