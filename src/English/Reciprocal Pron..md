## 句法作用
1. 直接宾语

| 序号  | 合并                                                | 分解                                                       | 指代的名词 |
| --- | ------------------------------------------------- | -------------------------------------------------------- | ----- |
| 1   | The students help one another/each other.         |                                                          |       |
| 2   | Health and cheerfulness mutually beget each other | Health beget cheerfulness, and cheerfulness beget health |       |

相互代词的含义：

2. 介词的宾语

| 序号  | 合并                                                             | 分解                                                        | 指代的名词 |
| --- | -------------------------------------------------------------- | --------------------------------------------------------- | ----- |
| 1   | For years the two sisters looked after each other/one another. | the older sister looked after younger sister, and the ... |       |


3. 所有格形式作定语

| 序号  | 合并                                                          | 分解  |
| --- | ----------------------------------------------------------- | --- |
| 1   | We should point out each other's shortcomings.              |     |
| 2   | They have great concern for one another's/each other's work |     |

作定语时表示“各自的”、“对方的”
## 用法
**关于指代的名词**
1. 作宾语时指代的名词（主语）必须是复数 或 具有 复数含义的词。
问：<span style="background:#fff88f">作定语时主语就可以是单数？</span>

**Reciprocal Pronouns 自身**
1. reciprocal pronouns 总是单数形式，包括所有格形式。即不能出现 each others 或 each others‘s 
2. each other 可以分开
- We each know what the other thinks.

3. one another 也可以分开构成习语
- one ... another

|                                       |     |
| ------------------------------------- | --- |
| They went from one shop to another.   |     |
| The books are piled one above another |     |

- one after another

|                                              |     |
| -------------------------------------------- | --- |
| They achieved independence one after another |     |

- one ... or another

|                                            |     |
| ------------------------------------------ | --- |
| I must have met him at one time or another |     |
