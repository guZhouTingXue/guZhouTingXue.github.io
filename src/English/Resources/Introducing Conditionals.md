## Mindmap
```markmap
# Conditionalss

## 作用<br>We use conditionals to show that something is true only when something else is true.

## Present real conditional

- 结构：
  when A happens, B happens.
- 示例：
  If it rains, I bring an umbrella
- 组成
  if clause: If it rains
  result clause: I bring an umbrella
- 句型变换：if clause 可以放在 result clause 后，两者之间不需要“，”
  - I bring an umbrella if it rains.
- 问题：
<br>1. A成立的条件下B成立 是否 隐含了 A不成的条件下 B也不成立？
<br>2. Present - 是if clause发生的时间 还是 result clause 发生的事件？
<br>3. real 指的是A 和 B之间的关系

## Present unreal conditional

- 结构：
  if A happened, B ==would== happen.
- example
  If I were you, I would take the job
- 动词的使用
<br>1. if clause 中 使用 simple past verb
<br>2. 如果if clause 中的动词是 be，那么固定使用 were
<br>3. result clause 中 使用 would + simple present verb

- 更多示例
  - If she were an animal, she would be a cat.

## Future real conditianl
- 结构：
 if A happens, B will happen.
- 示例
 If your plane is late, I will wait in the car.
- 使用情景
  - talk about possible situations in the future
  - threaten others

- 更多示例
  - If you let my daughter go now, that'll be the end of it.
```
