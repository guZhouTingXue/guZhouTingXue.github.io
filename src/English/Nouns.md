# 3. 名词 Nouns
>[!info]
>名词是用来表示人、物、地点或抽象概念等名称的词

## NOUN CLASSES

| 类别                                | 例词                                                                                                                                       | 说明                     |
| --------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------- | ---------------------- |
| Proper Noun                       | 地点：Britain, Canada, China<br>人：Chinese, Japanese<br>组织机构：The Academy of Science of China<br>节日：Thanksgiving Day<br>专有名称：The Yellow River | 1. 第一个字母必须大写。2. 介词可以小写 |
| Common Noun<br>->Countable Noun   |                                                                                                                                          |                        |
| Individual Noun                   | book, bus, computer                                                                                                                      |                        |
| Collective Noun                   | army, cattle                                                                                                                             |                        |
| Abstract Noun                     | beauty, idea, interest, success, thought                                                                                                 |                        |
| Common Noun<br>->Uncountable Noun |                                                                                                                                          |                        |
| Material Noun                     | coffee, cotton, gold, iron                                                                                                               |                        |
| Abstract Noun                     | fun, knowledge, length, love                                                                                                             |                        |
| 语言名词                              | Chinese, English, Japanese                                                                                                               |                        |
